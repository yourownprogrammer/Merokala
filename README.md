# Merokala

Merokala is a digital marketplace web application where users can browse and manage digital products.

## Features
- User registration and login
- Secure user authentication
- Product browsing
- Product search functionality
- Product upload and management
- Product detail view
- Simple and user-friendly interface
- Organized product listing
- Basic admin management
- Responsive design

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
